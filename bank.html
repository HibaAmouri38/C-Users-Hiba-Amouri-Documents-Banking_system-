Easily browse, edit and manage SQLite database inside your browser!

In order to work with this add-on, just open the app UI and drag a SQLite file to the designated area in the app

You can execute any SQLite command that you would like (SQL and SQLite have many differences but SQLite has most of the SQL standard).
